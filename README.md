# editor.js
Javascript实现的富文本编辑器

## 定义
富文本（rich text、formatted text或styled text），指具有风格、排版等信息的文本。  
段落上有段落样式，如对齐样式、行间距等；  
文字上有行内样式，如颜色、字体样式（黑体、斜体等）、字号、html特性（超链接等）。

编辑器指可以编辑信息的程序。

常见的富文本编辑器：Word、Google docs、有道云笔记、石墨文档、语雀等。

## 功能
- 编辑文字
- 编辑行内样式，例如：加粗、斜体、字体、颜色
- 编辑段落样式，例如：行间距、对齐、缩进等
- 撤销、重做
- 复制、剪贴、粘贴
- 打开、保存
